# Proyecto final Sistema operativos

En este proyecto se nos diocompleta libertad en como hacer la simulacion tanto sistema operativo, como lenguaje.

## Editor de codigo

Para este proyecto opte por hacer uso de Visual Studio, para asegurarte un correcto funcionamiento solo necesitas el archivo prueba.txt y saber donde esta ya que ahi estan los procesos a simular, puedes crear un mismo archivo de prueba con el nombre que quieres ya que el programa te da opcion a elegir el archivo.

>[!IMPORTANT]
>Sigue el formato del TXT: Nombre  TiempoInicio TamañoEnMemoria CiclosDeDuracion 

## Lenguaje de Programacion

Yo opte hacer esta practica en C++ ya que es el lenguaje mas usado durante la carrera y me encanta la libertad que te da y el control, ademas del rendimiento.

## Sistema operativo

Este proyecto solo funciona en windows ya que use la winApi para hacer la simulacion grafica, muchos compañeros optaron por mostrar el funcionamiento con caracteres ASCII por la terminal pero yo me impuse el reto de intentar aprender como funciona el entorno grafico de windows por encima.
