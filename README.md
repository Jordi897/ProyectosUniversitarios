# 📚 Proyectos de Ingeniería Informática – Universidad de Alicante

Este repositorio recopila los distintos **proyectos desarrollados durante la carrera de Ingeniería Informática** en la *Universidad de Alicante*. Como estudiante que ha optado por la **rama de Computación**, mi objetivo es mantener un historial organizado, accesible y útil tanto para mi propio aprendizaje como para cualquier persona interesada en revisar o reutilizar estos trabajos.

---

## 🎯 Objetivo del repositorio

El propósito principal de este repositorio es:

- Documentar proyectos académicos realizados a lo largo de la carrera.  
- Mantener una estructura clara que facilite la navegación entre cursos y asignaturas.  
- Mostrar la evolución de mis habilidades en programación, algoritmos, computación y desarrollo de software.  
- Servir como base para futuros proyectos personales y profesionales.

---

## 🗂️ Estructura del repositorio

El repositorio está organizado por **carpetas que representan el año del curso** en el que se realizaron los proyectos. Esto permite una visión cronológica del progreso académico.


Cada carpeta de curso contiene subcarpetas con las asignaturas y en ellas los proyectos, incluyendo código fuente, documentación y cualquier recurso adicional necesario.

Tambien se indicara cuando un proyecto a sido en conjunto y se hara las correspondientes menciones. Si no se indica nada es porque el proyecto se a hecho unicamente por mi.
